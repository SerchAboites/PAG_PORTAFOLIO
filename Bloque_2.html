GENERADOR DE DOCUMENTOS EN GOOGLE SHEETS

Esta herramienta surgió como respuesta a la necesidad de optimizar el proceso de creación de los PDF para las capacitaciones mensuales, el cual estaba consumiendo una cantidad considerable de tiempo. Después de explorar recursos en internet y encontrar algunos videos pertinentes sobre el tema, decidí profundizar en la aplicación de JavaScript para abordar esta problemática específica.

El objetivo era agilizar el proceso y reducir la carga de trabajo asociada con la generación de certificaciones mensuales para todo el personal. Para lograrlo, desarrollé fórmulas para extraer de manera eficiente los datos clave y empleé técnicas de programación para integrar estos datos en una plantilla predefinida, la cual contenía marcadores de texto.

Además, implementé estructuras de datos recursivas para asegurar que el proceso pudiera adaptarse dinámicamente al volumen de información presente en la base de datos. De esta manera, la herramienta pudo ejecutarse de manera óptima, independientemente del tamaño de los datos procesados.

ENVIO DE CORREOS MASIVOS CON GOOGLE SHEETS JAVASCRIPT

El departamento de calidad identificó un problema significativo en el proceso de recopilación de encuestas de calidad, que se realizaban de manera manual, lo que generaba obstáculos y demoras. Para mejorar esta situación, propusimos la implementación de Google Forms como una solución eficiente para recopilar la información necesaria. Sin embargo, surgió el desafío de encontrar una manera efectiva de enviar las encuestas a los clientes y registrar las respuestas.

Tras investigar a fondo, explorando posibles soluciones, surgió la idea de utilizar JavaScript para automatizar este proceso. Después de confirmar la viabilidad de esta opción y encontrar las herramientas y librerías necesarias, procedimos con la implementación. Esta estrategia incluyó el uso de HTML para evitar que los correos fueran clasificados como spam, mediante el almacenamiento de imágenes en la nube y la inclusión de enlaces directos en las encuestas enviadas.

Además, como parte de nuestra iniciativa para aumentar la participación de los clientes, ideamos la idea de sortear una tableta, lo que resultó en un aumento significativo en la cantidad de encuestas recopiladas. Logramos alcanzar nuestra meta en tan solo dos semanas, un logro notable considerando que en años anteriores este proceso podía llevar hasta tres meses.

Aprovechando la misma información recopilada, desarrollamos una solución para automatizar el cálculo de indicadores mediante gráficos y fórmulas. Esto eliminó la necesidad de recalcular y transcribir manualmente los datos, lo que a su vez ahorró tiempo y mejoró la precisión de nuestros análisis de calidad.
